And so not to forget this again.
  
This is how to clear the console screen in rails, type this,
  
In Ubuntu,
{% highlight ruby %}
system 'clear'
{% endhighlight %}

In Windows
{% highlight ruby %}
system 'cls'
{% endhighlight %}
